flex box를 이용해 화면의 가운데 정렬하기

body 안에 4개의 정렬값을 넣어주면 된다.

display:flex; /* 플렉스 컨테이너 지정*/
justify-content:center; /* 주축 정렬 중앙 */
align-items: center; /* 교차축 정렬 중앙 */
min-height: 100vh; /* 브라우저의 높이 100 다 사용 */

위 처럼 4개를 바디 속성에 주면, 바디 안에 정보들이 정가운데에 정렬된다.

gap 플랙스 항목 간에 간격을 주는 것

플랙스박스 컨데이너 안에 플랙스 항목들의 간격을 조절할 때 사용

gap: 15px;

플랙스 사이에 간격을 15px로 주는 것.
